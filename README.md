I worked on the **Booking Module**, implementing:
- booking creation and cancellation
- availability checks for selected date ranges
- booking retrieval by user

I also worked within a layered Spring Boot architecture using JPA repositories, DTO-based data flow, and Hibernate-managed entity relationships.

**Team repository:** https://github.com/ungureanuIrinuca/hotel-booking-system
